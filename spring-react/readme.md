## Description

Blogging website. Users can write articles and save to their account. 

### Features

- Account Management
- Browse existing blog pages
- Filter by tags
- Simple text editor
- Save to database

## Tools

- Spring Data Rest
- Thymeleaf
- JPA
- H2
- (Optional) Bootstrap

## Models

### Blog

- Author
- Page Content
- Create Date
- Tags

### User

- First Name
- Last Name
- Email Address
- Username
- Password
- Preferences
